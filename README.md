# 🚀 Fullstack React Strapi Blog 📚

Lagi belajar web development, khususnya di bagian **frontend**.  
Biar nggak cuma teori, saya coba bikin project **fullstack sederhana** sebagai latihan.

Dari project ini, *Alhamdulillah* saya belajar banyak — mulai dari bagaimana frontend dan backend bisa saling terhubung, sampai cara mengatur data dan tampilan pakai tools yang sering dipakai di industri.  
Masih jauh dari sempurna, tapi saya senang bisa menyelesaikannya dan mendapatkan banyak insight dari prosesnya. 😊

---

## 🧩 Tech Stack

- ✨ **Frontend**: React TypeScript, TailwindCSS, Redux  
- 🔌 **API Consumption**: React Query + Axios  
- 📦 **Backend**: Strapi (Headless CMS)  
- 📊 **Table UI**: TanStack React Table  

---

## 🧩 Fitur Utama

- 🔐 Login & Register  
- 📝 CRUD untuk Blog & Kategori  
- 👥 Halaman khusus untuk User  
- 📊 Halaman Dashboard Admin  

---

## 🎓 Hal yang Saya Pelajari dari Project Ini

- ✅ Pengelolaan state dengan Redux  
- ✅ Pengambilan & pengelolaan data API menggunakan React Query  
- ✅ Desain UI yang responsive dengan TailwindCSS  
- ✅ Menampilkan data dinamis dengan React Table  
- ✅ Dan tentu saja, konsistensi serta debugging 😅
